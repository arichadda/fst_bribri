# Finite State Transducer for Bribri


The Bribri language is spoken by approximately 7000 people in southern Costa Rica. It is part of the Chibchan language family, a family with languages in Costa Rica, Panama and Colombia. Bribri verbsuse suffixes to indicate different verbal tenses and aspects. Credit to Prof. Coto-Solano for this blurb and an introduction to the language. 

A transducer that takes Bribri words as input and gives you the separate morphemes as output.
```
Input: aleH 
Output: al-eH
```

A transducer that takes Bribri words as input and gives you the meaning of the morphemes as output.
```
Input:aleH
Output:cook-IPFV
```
